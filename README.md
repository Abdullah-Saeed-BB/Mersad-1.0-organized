<h1 align="center">مِداد</h1>
<h3 align="center">اولاً مِرصاد هو تطبيق لتنظيم وإدارة عمليات التصوير و كتابة المحتوى, ومِداد وهي إضافة مساعد ذكاء اصطناعي صمم لمساعدة المصورين والكُتاب في كتابة محتوى المرئي والمسموع.</h3>
<p align="center">تم استعمال نموذج <code>Gemini 3.1 Flash-Lite</code> كعقل لي مِداد.</p>
