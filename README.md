# مرصاد

مرصاد هو تطبيق ويب لإدارة جلسات التصوير الفيديوي: ينظّم المشاريع والمشاهد والمقاطع، يتتبّع تقدّم التصوير، ويوفّر teleprompter مدمجاً وأداة تشكيل للنصوص العربية. صُمّم للاستخدام الميداني في موقع التصوير — بيد واحدة، تحت ضغط الوقت — لفرق الإنتاج الاحترافية وصانعي المحتوى الأفراد.

الواجهة عربية أصيلة (RTL من الجذر، لا كترجمة)، بتصميم هادئ يخدم العمل ولا يستعرض نفسه.

## مكوّنات التطبيق

- `index.html` — التطبيق الرئيسي لإدارة المشاريع والمشاهد والمقاطع
- `teleprompter.html` — شاشة الملقّن (teleprompter)
- `tashkeel.html` — أداة تشكيل النصوص العربية
- `sw.js` و `manifest.json` — دعم PWA للعمل دون اتصال والتثبيت على الأجهزة
- `firebase.json` و `firestore.rules` و `storage.rules` — إعدادات الاستضافة وقواعد الأمان على Firebase

## ملاحظة حول الخطوط

الخطوط التجارية المستخدمة في النسخة الأصلية — **ثمانية** و**سلمى** — استُبعدت من هذا المستودع لأسباب ترخيصية. يتراجع التطبيق تلقائياً إلى خطوط النظام عند عدم توفرها.
